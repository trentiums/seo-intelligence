Output numbered action list sorted by highest impact + lowest effort first:

- Effort: Easy / Medium / Hard
- Impact: High / Medium / Low
- Category: Content / Technical / Schema / Links / Images / UX

Include quick wins section at top. End with ranking timeline estimate.
