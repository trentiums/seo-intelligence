---
description: Generates data-driven SEO content briefs from competitor analysis. Use when user asks for content ideas, blog post outlines, content plans, or what to write for a target keyword.
---

Use `search_serp` to find top competitors, crawl each with `analyze_page_seo`, then generate a content brief.

Output:
- Recommended title (keyword-rich, under 60 chars)
- Meta description template (150-160 chars)
- Heading structure (H1 → H2s → H3s)
- Target word count (competitor average + 10%)
- People Also Ask questions to answer
- Key topics competitors cover that must be included
- Recommended schema type
- Content gaps: what top rankers have that others miss
