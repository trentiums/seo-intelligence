---
description: Checks AI search engine citation visibility. Use when user asks about AEO, being cited in Perplexity, ChatGPT, or Google SGE, AI search rankings, or answer engine optimization.
---

Use `check_ai_search_visibility` MCP tool (requires PERPLEXITY_API_KEY and/or OPENAI_API_KEY in user config).

Check across Perplexity, ChatGPT/OpenAI, and Google SGE.

Output:
- Citation presence per platform: YES / NO / PARTIAL
- Sample queries where domain is cited
- Recommendations to improve AI citation rate:
  - Add Speakable schema
  - Improve E-E-A-T signals
  - Structured FAQ content
  - Authority link building
