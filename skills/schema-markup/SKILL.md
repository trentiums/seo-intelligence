---
description: Generates and validates JSON-LD schema markup. Use when user asks about structured data, rich results, rich snippets, FAQ schema, Product schema, or why a page has no rich results in Google.
---

Use `analyze_page_seo` to detect existing schema, then generate or fix JSON-LD markup.

Supported types: Article, BlogPosting, Product, LocalBusiness, FAQ, VideoObject, Speakable, BreadcrumbList.

Output:
- Complete JSON-LD block ready to paste into `<head>`
- Validation against Google Rich Results requirements
- Missing required fields flagged
- Optional recommended fields listed
- Preview of expected rich result appearance in Google Search
