Use `generate_ranking_plan` MCP tool on the provided URL and keyword.

Output numbered action plan sorted by highest impact + lowest effort first:
- Each action: what to change, where, specific implementation steps
- Effort: Easy / Medium / Hard
- Impact: High / Medium / Low
- Category: Content / Technical / Schema / Links / Images / UX
- Estimated implementation time

Summary: expected ranking improvement timeline based on keyword difficulty score.
